## Setup

1. Clone the reprository
```
git clone https://github.com/arnxv0/ymir.git
```
2. Setup frontend:
```
cd frontend
```
```
npm i
```
```
npm start
```

3. Setup backend:
```
cd backend
```
```
pip install -r requirements.txt
```
